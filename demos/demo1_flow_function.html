What to notice

  • The curve crosses zero at x = 0 — that is the fixed point.
  • Attractor: curve slopes downward at the crossing → arrows point inward → system returns here.
  • Repeller: curve slopes upward at the crossing → arrows point outward → system flees from here.
  • Steeper slope (higher k) = stronger pull (or push). Drag the particle to a new starting position.
